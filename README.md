# css-frequency-units

[![Build Status][ci-img]][ci]

List of CSS [`<frequency>`][frequency] units.

## Install

```sh
npm install css-frequency-units --save
```

## Usage

```js
var cssFrequencyUnits = require('css-frequency-units');
console.log(cssFrequencyUnits);
/* [
	"Hz",
	"kHz"
] */
```
